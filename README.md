---
### 프로젝트 세팅
yarn
```
yarn install
```

---
### 프로젝트 실행
```
yarn start
```

---
### 프로젝트 빌드
```
yarn build
```

---
### Git commit message
* feat : 새로운 기능에 대한 커밋  
* fix : 버그 수정에 대한 커밋
* build : 빌드 관련 파일 수정에 대한 커밋
* chore : 그 외 자잘한 수정에 대한 커밋
* docs : 문서 수정에 대한 커밋
* refactor :  코드 리팩토링에 대한 커밋
* test : 테스트 코드에 대한 커밋
